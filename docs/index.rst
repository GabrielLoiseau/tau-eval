**Tau-Eval** is a user-friendly, modular, and customizable Python library designed to benchmark and evaluate text anonymization algorithms. It enables granular analysis of anonymization impacts from both privacy and utility perspectives. Tau-Eval seamlessly integrates with `LiteLLM <https://www.litellm.ai/>`_ and `ü§ó Hugging Face <https://huggingface.co/>`_ to support a wide range of datasets, models, and evaluation metrics.
